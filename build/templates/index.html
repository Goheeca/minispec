This documentation tool is a cheat-sheet and a concept-navigation aid. Its goal is to help Common Lisp programmers quickly browse symbols for their definition and usage. The listings are example-driven, paired with concise descriptions, and providing readily available links to more comprehensive documentation.

The MiniSpec is still a Work in Progress. General features were included first, with more added as time allows.

Features

  • Easy to navigate. Symbols are grouped by concept while maintaining a searchable index.
  • Definitions are example-driven and can appear in multiple sections, if appropriate.
  • More comprehensive documentation is always accessible (just click the symbol name).
  • Includes popular libraries such as Alexandria, CL-Utilities, Iterate, and more.
  • Updating is simple. Fill in descriptions and examples using Markdown, or include a pet library in a local build.

Contribute?

Any help would be appreciated! For the most part, the front-end scaffolding is in place and there are a number of examples to convey the overall "vision". Aside from the style and organizational issues, a lot of the symbol definitions need to be filled out and existing ones reviewed for accuracy and clarity.

Here's the GitHub repo where you'll find the entries, in particular the sections and pages directories. And editing can even be done through the GitHub interface! See the readme file for more info.
