This documentation tool is a cheat-sheet and a concept-navigation aid. Its goal is to help Common Lisp programmers quickly browse symbols for their definition and usage. The listings are example-driven, paired with concise descriptions, and providing readily available links to more comprehensive documentation.

The MiniSpec is still a Work in Progress. General features were included first, with more added as time allows. Contribute to the project if you can.

Features

  • Easy to navigate. Symbols are grouped by concept while maintaining a searchable index.
  • Definitions are example-driven and can appear in multiple sections, if appropriate.
  • More comprehensive documentation is always accessible (just click the symbol name).
  • Includes popular libraries such as Alexandria, CL-Utilities, Iterate, and more.
  • Updating is simple. Fill in descriptions and examples using Markdown, or add your own pet library to a local build.
